NormalizeComments
================================

Normalize comments.

.. |FORMATTERNAME| replace:: NormalizeComments
.. include:: enabled_hint.txt

Normalizes spacing after the beginning of the comment. Fixes ``missing-space-after-comment`` rule violations
from the Robocop.
